I built an AI-powered Banking Assistant from scratch using Llama 4 + Groq!

Introducing Apex Banking AI Assistant - an intelligent chatbot that handles real banking operations using AI tool-calling:

What it can do:
- Check account balances & status
- View transaction history
- Check loan eligibility (personal, home, auto, business)
- Report & instantly block lost/stolen cards
- Find nearest bank branches & ATMs

Tech stack:
- AI: Llama 4 Scout (Meta's latest model)
- Inference: Groq (ultra-fast AI inference)
- Backend: Python serverless functions
- Frontend: Custom-built chat UI
- Deployment: Vercel

The AI doesn't just chat - it uses function calling to execute real banking tools and return structured data to users. This is how modern AI agents work in production.
